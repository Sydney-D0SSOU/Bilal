## Où les télécharger / mettre à jour ?

1. **Depuis Figma** (recommandé, pas d’expiration)  
   Ouvre le fichier [MON-SITE](https://www.figma.com/design/f9k7xepd854eySpP1c1FaU/MON-SITE), frame **2082-1696**.  
   Sélectionne chaque logo → **Export** (PNG @2x ou **SVG** pour le premier) → enregistre ici avec les noms ci-dessus.

2. **Script / Cursor**  
   On peut regénérer les PNG via l’API Figma MCP (`get_design_context` sur `2082-1696`) puis `curl` des URLs `https://www.figma.com/api/mcp/asset/...` — ces liens **expirent (~7 jours)**.

Ne supprime pas ces fichiers du dépôt : sans eux, les logos ne s’affichent pas.
